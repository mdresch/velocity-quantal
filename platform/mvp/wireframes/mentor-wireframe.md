# Mentor Portal Wireframe

Purpose
- Provide lightweight coaching, review, and endorsement capabilities for mentors and advisors.

Primary Goals
- Quickly view assigned pilots and summaries.
- Leave private mentorship notes and public endorsements.
- Recommend improvements and mark priority suggestions.

Main Sections
1. Mentor Dashboard
   - Assigned pilots list (status badges: onboarding, active, submitted, reviewed)
   - Quick metrics: time-saved estimates, pilot progress
2. Pilot Detail View
   - Summary card (problem statement, KPIs, owner)
   - Evidence snapshot (thumbnails, counts)
   - Mentorship notes panel (private) + endorsements (public)
   - Suggested action buttons: "Request Clarification", "Flag for Review", "Endorse"
3. Notifications
   - Mentorship requests, submission updates, scheduled office hours
4. Profile & Availability
   - Set office hours, accept mentee requests, link to calendar

Interactions
- Clicking "Endorse" opens modal to write short endorsement and optionally create a public case-study prompt.
- Mentors can tag evidence items for follow-up and assign micro-tasks to Pilot Owners.

Accessibility
- Keyboard-first navigation, clear focus states, accessible modals.
