# University Student / Researcher Portal Wireframe

Purpose
- Provide sandboxed, consented access to anonymized datasets and learning materials; allow students to participate in annotation tasks and supervised research contributions.

Primary Goals
- Discover usable datasets and pilot projects accepting contributors.
- Complete annotation tasks and submit results with provenance.
- Access learning modules and office hours with mentors.

Main Sections
1. Student Dashboard
   - Active tasks, available pilots, learning progress
2. Task / Annotation View
   - Evidence viewer with annotation tools (highlight, comment, label)
   - Submission box and progress tracker
3. Learning Resources
   - Tutorials, sample prompts, reproducible starter kits
4. Mentorship & Events
   - Sign up for office hours, view recommended mentors

Privacy & Consent
- Students only access anonymized data; consent flows recorded; contributed annotations are versioned and attributable.

Accessibility
- Clear contrast, keyboard navigation, and tutorials for first-time contributors.
