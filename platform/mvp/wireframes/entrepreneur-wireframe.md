# Entrepreneur / Founder Portal Wireframe

Purpose
- Fast path for solo founders and micro-teams to run pilots, upload evidence, and produce investor or partner-facing export bundles.

Primary Goals
- Rapidly create a pilot workspace, upload evidence, and track KPIs.
- Produce a neat export (zip + summary) for partners or investors.
- Access mentorship or partner referral flows quickly.

Main Sections
1. Quick Start Dashboard
   - Create pilot button, recent uploads, KPI snapshot (time saved, validation rate)
2. Upload & Ingest Flow
   - Drag-and-drop evidence, ingest status, auto-chunking preview
3. Submission Builder
   - Select evidence, add cover note, choose partner program, preview package
4. Export & Share
   - Download zip, generate share link, request partner review
5. Growth Tools
   - Templates for pitches, one-click investor brief, integration guides

UX Notes
- Minimal fields, inline help, guided template for first-time founders.
- Mobile-first flows for on-the-go uploads.
