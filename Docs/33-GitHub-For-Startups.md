## Offer Summary

GitHub for Startups helps founders and developer teams accelerate product development and growth through educational content, events, partner deals, and a founder-focused ecosystem. The program combines onboarding resources, community engagement, and promotional opportunities.

## Program highlights
- Educational content, workshops, and a monthly newsletter.
- Networking and community events (local and virtual).
- Opportunity to be featured in GitHub marketing assets (case studies, podcasts, videos) via application.
- Program resources and an event calendar (subscribe to the program calendar).
- Support channels: program email startups@github.com, application portal, and documentation links.

## What the program offers
- Application & onboarding to the GitHub for Startups program.
- Resources: getting started guides, documentation, and curated repo of program resources.
- Event calendar (Luma-powered) for program events and community activities.
- Support to host startup community events at GitHub offices (request via program contact).

## Featured & promotional opportunities
- Apply to be featured in GitHub case studies or media (separate interest form; selection not guaranteed).
- Sponsorship and podcast/video participation opportunities for qualifying startups.

## Eligibility & verification
- Eligibility varies by program cohort; typical checks include founder/company details, funding status, and partner affiliation.
- GitHub may request verification such as a Crunchbase profile, investor contact, or partner confirmation.

## How to apply
1. Visit the GitHub for Startups application portal and fill the form (corporate email, website, company details, funding/partner info).
2. Monitor email for follow-up questions from the program team.
3. Provide requested verification documents if asked.

## FAQ (short)
- What resources do I get? Education, events, community, featured opportunities, and partner offers.
- Can I be featured? Yes — submit the interest form; GitHub will review and reach out if appropriate.
- How are offers verified? Program team reviews applications and may request supporting evidence.

## Recommended evidence checklist for applications and partner submissions
- Corporate email and workspace domain.
- Company website or public profile (Crunchbase, PitchBook, AngelList).
- Funding or partner affiliation proof (investor contact or accelerator reference).
- Short justification: product fit for GitHub ecosystem and goals for the program (marketing, developer adoption, fundraising support).
