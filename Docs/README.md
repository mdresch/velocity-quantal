## How to contribute

- Create a branch named `docs/<artifact>-short-desc-YYYYMMDD` or the single PR branch `docs/split-opportunities-YYYYMMDD`.
- Update front-matter `version` and `status` in the file.
- Add an entry to `CHANGELOG.md` when merging.
