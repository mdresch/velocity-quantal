## Purpose

Define KPIs, reporting cadence, and dashboards to assess pilot and program performance.

## Suggested KPIs

- Number of opportunities identified.
- Number of pilots executed.
- Capital generated (gross) per pilot and program aggregate.
- Pilot success rate (% meeting acceptance criteria).
- Time-to-validation (weeks).
- Mentor engagement hours and feedback adoption.

## Reporting

- Monthly program dashboard: aggregated KPIs and top insights.
- Pilot report: per-pilot metrics and retrospective notes.
